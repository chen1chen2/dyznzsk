# 项目管理需求文档 (PRD)

## 7. 详细功能说明

### 7.5 项目智能问答联动

#### 7.5.1 打开智能问答抽屉

| 字段     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 功能编号 | PRO-15 / PRO-16                                              |
| 功能描述 | 用户在文件或任务页签打开智能问答抽屉，并基于项目关联文档提问 |
| 前置条件 | 用户已进入项目详情，拥有问答权限                             |
| 优先级   | P0                                                           |

**页面元素**：

| 元素         | 类型     | 说明                    | 校验规则                                                                                                                                                                                                                                                                                |
| ------------ | -------- | ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 智能问答     | 按钮     | 位于文件/任务批量操作区 | ***文件/任务页签无选中项时可点击划出智能问答（此时为新建智能问答，不关联任何文档）；<br />勾选文件，点击智能问答（此时为新建智能问答，关联选中文档与对应知识库）；***<br /><br />***全部任务、我的任务、可领取任务都提供勾选任务，点击智能问答，右侧划出智能问答弹窗*** |
| 智能问答抽屉 | 右侧抽屉 | 嵌入完整问答工作区      | 抽屉打开后不重置项目页面状态                                                                                                                                                                                                                                                            |
| 关闭按钮     | 图标按钮 | 关闭抽屉                | ***关闭后清空文件/任务的勾选状态***                                                                                                                                                                                                                                             |
| 进入完整页面 | 图标按钮 | 跳转完整智能问答页      | 跳转后保留关联上下文                                                                                                                                                                                                                                                                    |
| 知识库按钮   | 按钮     | 展示已选知识库数量      | 默认应关联项目知识库或文档快翻知识库                                                                                                                                                                                                                                                    |

**交互逻辑**：

1. 用户在文件页签或任务页签勾选稿件。
2. 用户点击“智能问答”。
3. 系统从页面右侧打开智能问答抽屉。
4. ***若用户已勾选文件或任务，系统自动关联对应项目文档与对应知识库；若译文已生成，应同时关联原文和译文。***
   1. ***用户如果没有勾选文件或任务，开智能问答抽屉，新建一个空白的会话***
5. 抽屉打开后，用户可使用快捷问题“知识图谱”“文件概述”“翻译建议”，或直接输入问题。
6. 用户关闭抽屉后，项目详情页保持原页签、筛选、分页***，需要清空勾选状态。***

**异常处理**：

| 异常场景                                                    | 处理方式                                                                                                    |
| ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| 选中文档尚未解析完成（文档在知识库没有解析完毕，或解析失败) | ***点击智能问答，toast提示：文档智能解析中，请等待解析完成***                                       |
| 选中的文档不在知识库                                        | ***文案为：“知识库暂无当前文档，未关联文档"<br />提示后可正常打开智能问答弹窗，只是没有自动关联知识库*** |

#### 7.5.2 在线笔译侧栏智能问答

| 字段     | 说明                                                                                           |
| -------- | ---------------------------------------------------------------------------------------------- |
| 功能编号 | PRO-19 / PRO-20                                                                                |
| 功能描述 | 用户在在线笔译页面右侧栏切换“智能问答”页签，并可点击页签文字旁的外链图标新开完整智能问答页面 |
| 前置条件 | 用户已从项目文件或任务进入在线笔译页面，且拥有智能问答权限                                     |
| 优先级   | P0 / P1                                                                                        |

**页面元素**：

| 元素           | 类型     | 说明                                                    | 校验规则                                             |
| -------------- | -------- | ------------------------------------------------------- | ---------------------------------------------------- |
| 智能问答页签   | Tab      | 位于在线笔译页面右侧栏，与提示、QA检查、备注、日志并列  | 文案固定为“智能问答”                               |
| 外链图标       | 图标按钮 | 位于“智能问答”文字旁，用于新开完整问答页              | 使用通用外链图标；hover 提示“新开页面进入智能问答” |
| 右侧嵌入问答区 | 内容区域 | 在在线笔译右侧栏内嵌入智能问答工作区                    | 选中“智能问答”页签时展示；宽度应扩展到可用问答宽度 |
| 完整智能问答页 | 独立页面 | 新浏览器页面打开完整智能问答工作区                      |                                                      |
| WebCat 入口    | 文字按钮 | 项目文件列表展示“查看”，任务列表展示“在线笔译/预览” | 点击后进入对应稿件的 WebCat 页面                     |

**交互逻辑**：

1. 用户进入在线笔译页面后，右侧栏展示“提示、QA检查、备注、智能问答、日志”等页签。
2. ***用户点击“智能问答”页签，系统在右侧栏内嵌入智能问答工作区。***
   1. ***智能问答默认勾选了webcat对应任务的文档，可能对应多个文档***
   2. ***进入webcat页面，首次打开智能问答tab，新建一个会话并将知识库和文档进行关联，后续打开智能问答tab都不新建会话，保留之前的会话内容***
3. 用户点击“智能问答”文字旁的外链图标，系统使用新浏览器页面打开完整智能问答页。
4. 外链点击不应改变当前在线笔译页面的页签、编辑内容、筛选状态和任务上下文。
5. 用户在项目文件列表点击“查看”，或在任务列表点击“在线笔译/预览”时，系统进入对应文档的 WebCat 页面。

### 7.6 文件概述与知识点

#### 7.6.1 自动生成与预览

| 字段     | 说明                                                               |
| -------- | ------------------------------------------------------------------ |
| 功能编号 | PRO-08                                                             |
| 功能描述 | 文件进入项目后，系统生成概述和知识点，辅助项目理解和问答上下文检索 |
| 前置条件 | 文件已上传或任务已生成，文件解析完成                               |
| 优先级   | P1                                                                 |

**页面元素**：

| 元素           | 类型       | 说明                                                                                                           | 校验规则                            |
| -------------- | ---------- | -------------------------------------------------------------------------------------------------------------- | ----------------------------------- |
| 概述生成中图标 | 状态图标   | ***项目详情-文件列表、项目详情-任务(全部任务、我的任务、可领取任务)的文件名旁展示“智能分析中”状态*** | hover 提示“概要与知识点生成中...” |
| 概述完成图标   | 状态图标   | ***项目详情-文件列表、项目详情-任务(全部任务、我的任务、可领取任务)的文件名旁展示完成状态***           | 图标占位稳定，不造成表格跳动        |
| 概述浮层       | Hover 浮层 | ***项目详情-文件列表、项目详情-任务(全部任务、我的任务、可领取任务)的展示概述正文和知识点标签***       | 浮层不遮挡关键操作按钮              |
| 知识点标签     | 标签       | 展示文件主题、术语或业务要点                                                                                   | 标签过多时折叠展示                  |

**交互逻辑**：

1. 文件进入项目后，系统创建概述与知识点生成任务。
2. 生成中时，文件名旁展示 loading 状态。
3. 生成完成后，文件名旁展示完成图标。
4. 用户 hover 文件名或图标时，系统展示概述和知识点。

### 7.7 项目文档内容同步至知识库

#### 7.7.1 内容更新提示与入口

| 字段     | 说明                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 功能编号 | PRO-KBSYNC-01                                                                |
| 功能描述 | 系统检测项目文档的原文或译文内容更新，支持用户将更新后的内容同步至项目知识库 |
| 前置条件 | 文档已进入项目知识库，且原文或译文较知识库中当前版本已发生更新               |
| 优先级   | P0                                                                           |

**页面元素**：

| 元素         | 类型     | 说明                                                                                    | 校验规则                                   |
| ------------ | -------- | --------------------------------------------------------------------------------------- | ------------------------------------------ |
| 内容更新标签 | 胶囊标签 | 位于项目概览区“添加历史语料库”标签后，展示“发现 N 份文档内容已更新，可同步到知识库” | 使用浅青色胶囊标签样式；N 为待同步文档数量 |
| 更新数量     | 普通文本 | 展示待同步文档数量 N                                                                    | 不可点击，不提供 hover 交互                |
| 查看         | 文字按钮 | 位于标签内，点击打开“待同步文档”弹窗                                                  | 每次打开时重置搜索、页码和勾选状态         |
| 同步中标签   | 胶囊标签 | 同步任务发起后展示“文档内容同步中...”                                                 | 不再展示“查看”入口，避免重复发起         |

**交互逻辑**：

1. 存在待同步文档时，项目概览区展示浅青色内容更新标签。
2. 数量 N 仅作为普通文本展示，用户仅可通过“查看”进入待同步文档弹窗。
3. 用户成功发起同步后，页面提示“已发起文档内容同步”，标签切换为“文档内容同步中...”。

#### 7.7.2 待同步文档弹窗

| 字段     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 功能编号 | PRO-KBSYNC-02                                                    |
| 功能描述 | 用户可搜索、分页、勾选内容已更新的项目文档，并发起知识库内容同步 |
| 前置条件 | 存在至少 1 份待同步文档                                          |
| 优先级   | P0                                                               |

**页面元素**：

| 元素       | 类型     | 说明                                                              | 校验规则                                              |
| ---------- | -------- | ----------------------------------------------------------------- | ----------------------------------------------------- |
| 弹窗标题   | 标题     | 展示“待同步文档”                                                | 固定文案                                              |
| 弹窗副标题 | 说明文案 | 展示“发现 N 份文档内容已更新，可将原文与译文(如有)同步至知识库” | N 为待同步文档总数                                    |
| 文件名搜索 | 输入框   | 按文件名称模糊筛选列表                                            | 输入后自动回到第 1 页；无结果时展示“未找到匹配文档” |
| 文件列表   | 表格     | 展示勾选框、文件名、文件密级、更新时间                            | 文件名超长时单行省略，hover 展示完整名称              |
| 全选       | 复选框   | 勾选或取消勾选当前页文档                                          | 仅影响当前页，不改变其他页已选状态                    |
| 文件勾选框 | 复选框   | 选择需要同步的文档                                                | 弹窗每次打开时默认全部不勾选                          |
| 分页       | 分页器   | 展示总条数、页码、上一页、下一页和每页条数                        | 默认每页 5 条；切换页码保留其他页勾选状态             |
| 已选数量   | 辅助文案 | 弹窗底部展示“已选 N 份”                                         | 跨页累计已选数量                                      |
| 取消       | 次要按钮 | 关闭弹窗且不发起同步                                              | 关闭后不保留当前搜索、页码和勾选状态                  |
| 同步       | 主按钮   | 将已勾选文档的原文与译文（如有）同步至知识库                      | 未勾选文档时禁用；已勾选至少 1 份文档时可点击         |

**交互逻辑**：

1. 弹窗每次打开时，搜索关键词为空、页码为 1、所有文档默认不勾选。
2. 用户可逐条勾选文档，也可通过表头复选框勾选当前页所有文档。
3. 搜索条件变化后自动回到第 1 页；切换页码时保留已选文档。
4. 未勾选任何文档时，“同步”按钮禁用。
5. 用户勾选文档并点击“同步”后，弹窗关闭，系统提示“已发起文档内容同步”，页面标签切换为“文档内容同步中...”。
6. 同步内容必须包含所选文档的原文；若译文已生成，同时同步对应译文。

**异常处理**：

| 异常场景   | 处理方式                                             |
| ---------- | ---------------------------------------------------- |
| 未勾选文档 | “同步”按钮禁用，不发起请求                         |
| 搜索无结果 | 列表展示“未找到匹配文档”，总条数展示 0             |
| 重复提交   | 首次提交后关闭弹窗并切换同步中状态，阻止重复发起     |
| 同步失败   | 保留待同步状态，提示失败原因，允许重试               |
| 文档无译文 | 仅同步原文，不影响本次任务的成功状态                 |
| 无同步权限 | 不展示标签和“知识库同步”入口；后端必须拒绝越权请求 |

#### 7.7.3 项目文件更多操作

| 字段     | 说明                                                 |
| -------- | ---------------------------------------------------- |
| 功能编号 | PRO-KBSYNC-03                                        |
| 功能描述 | 将次要文件操作收纳至三点菜单，并支持单文档知识库同步 |
| 前置条件 | 用户对项目文件具有对应操作权限                       |
| 优先级   | P0                                                   |

**页面元素**：

| 元素         | 类型     | 说明                                                                       | 校验规则                                                             |
| ------------ | -------- | -------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| 查看         | 文字按钮 | 进入文档对应的 WebCat 页面                                                 | 保留当前项目上下文                                                   |
| 导出         | 文字按钮 | 导出当前文档                                                               | 沿用原项目文件导出规则                                               |
| 更多操作     | 图标按钮 | 使用三点图标展开下拉菜单                                                   | 需提供 `aria-label` 和 hover 提示                                  |
| 更多操作菜单 | 下拉菜单 | 依次展示“拆分”、“质量评估”、“知识库同步”、“上传离线文件”、“删除” | “知识库同步”位于“质量评估”下方；“删除”与上方操作使用分割线隔开 |

**交互逻辑**：

1. 用户点击三点图标后，展开文件更多操作菜单。
2. 菜单展示顺序为：拆分、质量评估、知识库同步、上传离线文件、删除。
3. 用户点击“知识库同步”后，菜单关闭，系统提示“已发起文档内容同步”。
4. 用户点击菜单外部或菜单失去焦点时，菜单关闭。
5. “删除”为危险操作，应使用危险色并沿用原删除确认流程。

### 7.8 新增需求验收要点

| 验收编号  | 验收场景                 | 预期结果                                                                  |
| --------- | ------------------------ | ------------------------------------------------------------------------- |
| PRO-AC-01 | 项目文档存在内容更新     | 项目概览区展示浅青色胶囊标签，数量为普通文本，“查看”可点击              |
| PRO-AC-02 | 打开待同步文档弹窗       | 搜索词为空、页码为 1、所有文档默认不勾选，“同步”按钮禁用                |
| PRO-AC-03 | 搜索和分页勾选待同步文档 | 搜索后回到第 1 页；全选仅影响当前页；跨页已选数量正确累计                 |
| PRO-AC-04 | 勾选文档并点击同步       | 弹窗关闭，提示“已发起文档内容同步”，顶部标签切换为“文档内容同步中...” |
| PRO-AC-05 | 打开项目文件更多操作     | 按固定顺序展示拆分、质量评估、知识库同步、上传离线文件和删除              |
| PRO-AC-06 | 点击单文档“知识库同步” | 更多菜单关闭，提示“已发起文档内容同步”                                  |
| PRO-AC-07 | 点击项目文件“查看”     | 进入该文档对应的 WebCat 页面，项目上下文正确                              |
