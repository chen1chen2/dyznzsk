# 知识库问答 Feature List

## 模块1：AI 问答工作台

| 功能编号 | 功能名称 | 功能描述 | 优先级 | 备注 |
|---------|---------|---------|--------|------|
| CHT-01 | 新建对话 | 用户可从侧边栏或顶部返回入口创建新对话，用于发起新的问答主题 | P0 | 当前前端已实现草稿态 |
| CHT-02 | 历史会话切换 | 用户可点击历史会话切换当前会话内容 | P0 | 当前为前端内存数据 |
| CHT-03 | 问题输入 | 用户可在输入框输入问题，系统展示 0/5000 字数统计 | P0 | 占位文案“在这里向小译提问~”，最大 5000 字 |
| CHT-04 | 发送问题 | 用户输入非空问题后可点击发送或按 Enter 提交，系统生成或追加会话 | P0 | 空内容禁用发送，Shift+Enter 换行 |
| CHT-05 | AI 回答展示 | 系统以对话流展示 AI 回答、回答完成状态和回答操作 | P0 | 当前为静态示例回答 |
| CHT-06 | 回答操作 | 用户可对回答执行复制操作 | P1 | 当前仅保留复制入口 |
| CHT-07 | 知识库开关 | 用户可看到已选择知识库数量，并打开知识库选择弹窗 | P0 | 当前默认选中 1 个知识库 |
| CHT-08 | 互联网开关 | 用户可开启或关闭互联网检索能力 | P1 | 当前已实现前端开关态 |
| CHT-09 | 附件入口 | 用户点击附件按钮后先唤起系统文件选择器，选择文件后进入上传弹窗 | P0 | 已实现文件选择后打开上传弹窗 |
| CHT-10 | 重新开始 | 用户可从当前会话重新开始新的问答 | P1 | 当前进入新建对话态 |
| CHT-11 | 附件上传与密级设置 | 用户可上传最多 5 个附件，逐个或批量设置涉密等级，设置完成后自动上传 | P0 | 展示进度条、上传完成标识、超限 toast，发送后附件移动到问题上方 |

| 功能编号 | 功能名称 | 功能描述 | 优先级 | 备注 |
|---------|---------|---------|--------|------|

## 完整性检查

| 检查项 | 当前结论 | 产品化建议 |
|-------|---------|-----------|
| 问答是否真实生成 | 当前为前端静态示例 | 接入问答/RAG 服务 |
| 会话是否持久化 | 当前为前端内存态 | 接入会话与消息 API |
| 引用是否可追溯 | 当前为 mock 命中文件 | 后端返回引用文件、切片 ID、段落范围和预览 URL |
| 文件标签是否完整 | 当前有密级和语种标签 | 后端强制返回文件密级与语种，缺失时按异常处理 |
| 切片是否完整高亮 | 当前按关键词模拟整段高亮 | 后端返回准确切片范围，避免误判 |
| 权限是否强校验 | 当前只有前端提示 | 后端按人员密级过滤并审计 |
| 文档预览是否真实 | 当前为示例段落 | 接入文档解析和原译对照渲染服务 |
| 附件上传是否真实 | 当前为前端模拟上传进度 | 接入文件上传服务、失败重试、病毒扫描和涉密等级后端校验 |
